# FOREX - Forest Expedition

Sitio web estático para FOREX, un operador turístico enfocado en experiencias de naturaleza y atractivos de Jauja, Perú.

## Descripción

El proyecto presenta una página principal con navegación hacia las secciones:

- Quiénes somos
- Lugares turísticos
- Rutas
- Contactos

También incluye páginas internas para destinos como:

- Santuario de los Walis
- Tunanmarca
- Laguna de Paca

## Tecnologías

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## Estructura del proyecto

- `index.html`: página principal del sitio.
- `css/`: estilos globales del proyecto.
- `js/`: scripts, actualmente con la interacción del menú principal.
- `img/`: recursos gráficos de la portada.
- `somos/`: sección de quiénes somos.
- `rutas/`: sección de rutas turísticas.
- `contactos/`: formulario y datos de contacto.
- `lugares/`: páginas de lugares turísticos, con subcarpetas para cada destino.

## Uso local

Al ser un proyecto estático, no requiere instalación de dependencias.

1. Abre `index.html` directamente en el navegador, o
2. Ejecuta el sitio desde un servidor local como XAMPP, Live Server o cualquier servidor web simple.
